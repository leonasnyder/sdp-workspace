# SDP Workspace — Independent Facilitator App

A free tool for Independent Facilitators in California's Self-Determination Program (SDP).
Track clients, budgets, spending plans, goals, compliance, PCPs, forms, and documents —
all in the browser.

## Privacy
All data stays in the user's own browser (localStorage). Nothing is sent to any server.
Each user can connect their own backup folder (Chrome/Edge) — e.g. Google Drive for
Desktop — for automatic local backups.

## Deploy
This is a single static file. Deploy `index.html` to any static host (Vercel, Netlify,
GitHub Pages). No build step, no dependencies to install.

## Browser support
Best in Chrome or Edge (folder auto-backup, document filing). Works in Safari/Firefox
with manual backup/restore.
